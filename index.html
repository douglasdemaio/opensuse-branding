🦎 Adaptive Chameleon

One Lizard,
Many Colors

The openSUSE brand is built on a single idea: the chameleon adapts. The mascot stays the same — recognizable, friendly, distinctly ours — but its color shifts to match the personality of each sub-project. Tumbleweed gets its own hue, Leap gets another, MicroOS yet another. The shape is constant; the palette is alive.

This philosophy lets every project feel distinct while remaining unmistakably part of the openSUSE family. A user who knows the chameleon from Tumbleweed will instantly recognize it on Kalpa — just in a different coat. The identity is modular by design: colors, gradients and accents change; the structure and spirit stay fixed.

How it works

The parent distributions supply the package base, build infrastructure and QA pipeline. Specialized branches inherit all of this and layer on their own identity: MicroOS adds immutability and transactional updates, Kalpa wraps that in a Plasma desktop, and Slowroll applies a delayed-update cadence to Tumbleweed's rolling stream. Each branch uses its own Color DNA pair — visually communicating its lineage and its unique purpose.

Asset Guidelines

Rules for producing consistent physical and digital brand assets across the ecosystem.

Sticker Guidelines

All stickers use a 3mm Bagel Beige (#fff8ee) border around the artwork. This gives die-cut stickers a clean, premium edge and ensures the chameleon has breathing room on any surface.

📏 3mm border 🎨 #fff8ee beige ✂️ Die-cut ready 📐 24px corner radius 🖨️ CMYK safe

Wallpaper Guidelines

Desktop wallpapers use low-poly gradient backgrounds built from the sub-project's Color DNA pair. The chameleon silhouette may appear as a subtle watermark. Gradients flow bottom-left to top-right.

🔺 Low-poly mesh 🌈 2-color gradient ↗️ BL → TR flow 🖥️ 3840×2160 min 🦎 Watermark optional

Quick Reference

Sticker Production

  • Border: 3mm, Bagel Beige (#fff8ee)
  • Corners: 24px radius (scaled to print size)
  • Min size: 40mm × 40mm
  • Bleed: 2mm beyond border
  • Format: SVG source, PDF print-ready

Wallpaper Production

  • Style: Low-poly triangulated gradient
  • Colors: Sub-project Color DNA pair
  • Gradient direction: bottom-left → top-right
  • Resolution: 3840×2160 (4K) minimum
  • Formats: PNG (lossless) + SVG (source)

Typography

  • Primary: SUSE (headings, emphasis)
  • Fallback: system sans-serif stack
  • Body weight: 400 (Regular)
  • Heading weight: 700 (Bold)
  • Line height: 1.65 (body), 1.2 (headings)

Branding in Action

How the adaptive chameleon system comes to life across real-world touchpoints. Each project maintains its unique identity while staying unmistakably openSUSE.

Laptop Stickers

Die-cut stickers with the Bagel Beige border. Each project logo is instantly recognizable at conference tables and hackathons — a personal badge of which openSUSE flavor you run.

Desktop Wallpapers

Low-poly gradient backgrounds in each project's Color DNA. The logo appears as a subtle watermark — bold enough to identify, quiet enough for daily use.

Conference Swag

T-shirts, hoodies, and merch. The logo works in single-color on dark fabrics, full-color on light ones. Each sub-project can have its own colorway while keeping the family connection.
